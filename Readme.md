## Setup

To set up the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Menzy/alx-backend.git
    ```
2. Navigate to the project directory:
    ```bash
    cd alx-backend
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

To start the server, run:
```bash
npm start
```
